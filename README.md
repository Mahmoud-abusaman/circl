# Circl Mentorship Platform

Welcome to the Circl Mentorship Platform! This is a dual-sided mentorship application built with Django, Jinja2, and Django Channels.

## Features
- **Dual-sided Mentorship:** Users can both teach and learn.
- **Search & Discovery:** Find mentors easily.
- **Meeting Lifecycle:** Manage sessions from request to completion.
- **User Profiles:** Detailed user profiles including skills, experience, and interests.
